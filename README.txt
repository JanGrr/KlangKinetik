KlangKinetik - jangrr.github.io/KlangKinetik

Audio-Datei & 360° Image zu erstellen funktionen leider nur mit Server
deswegen am besten einfach jangrr.github.io/KlangKinetik besuchen 
oder Node.js installieren und http-server damit installieren
  Node.js Download: https://nodejs.org/en/download/
  im KlangKinetik-Ordner die Befehle: 
    1. npm install -g http-server
   (2. evtl Skript-Ausführung deaktiviert -> Powershell mit Administrator -> Befehl: Set-ExecutionPolicy RemoteSigned)
    3. http-server
